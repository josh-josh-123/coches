# coches


1. Título del proyecto

   Proyecto7 con dataset de coches de segunda mano vendidos en USA

2. Descripción del proyecto

    El proyecto tiene por objeto publicar una página web donde se muestren graficas que den cuenta de variables relevantes del mercado de venta de coches en USA
   
3. Funcionalidades

Histograma "Distribución de precios de vehículos"
    - Boton para bines amplios
    - Botón para bines angostos
Gráfica de dispersión "Relación entre año y precio:
    - Botón para gráfica de todas las marcas
    - Botón para gráfica de Toyota
    - Botón para gráfica de Chevrolet

4. Instrucciones de instalación

    - Entorno virtual: Se debe crear un EV para ejecutar una versión de Pyton que sea compatible con streamlit (Python desde 3.7 hasta 3.13)

    - Librerías a instalar: pandas (2.3.3) - Para análisis de datos , plotly (6.5.0) - Para visualizaciones interactivas ,  streamlit (1.52.2) - Para la aplicación web , nbformat (5.10.4) - Para Jupyter notebooks  , python (3.13.11) - Versión compatible  con streamlit
    
    - Comando para ejecutar la aplicación: Desde el entorno virtual creado, ejecutar:
        streamlit run app.py
